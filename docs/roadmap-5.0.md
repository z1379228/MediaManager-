# MediaManager 5.0 候選更新列表

## P0：搜尋強化

- 在 UI 加入搜尋來源選擇器、跨 MOD 分頁與各來源健康狀態。
- 以標題、歌手、時長、語言、類別建立本機排序模型，支援相似與替代搜尋。
- 縮圖、音訊試聽及可選影片預覽共用快取，限制容量並提供一鍵清除。
- 加入中文／日文／英文別名與拼字容錯，但不把搜尋紀錄送往第三方分析服務。

## P0：MOD 適配

- 提供 Search/Download Adapter SDK、範例 MOD、合約測試器與相容性報告。
- 支援 MOD 宣告網站域名、速率限制、必要工具與可選能力；啟動時只提示缺少的依賴。
- MOD 個別熔斷、逾時、資源配額及版本遷移，故障時不拖垮核心 UI。
- 優先完善 YouTube、Bilibili，再逐一驗證 Facebook、Instagram、Threads；未驗證網站不標示完整支援。

## P0：下載強化

- 分析後顯示實際可用畫質、編碼、容器、音軌、字幕與預估容量。
- 可續傳分塊下載、磁碟空間預留、斷線重試及程式重啟後復原。
- 播放清單／批量工作支援選集、命名預覽、重複偵測與個別失敗重跑。
- 長篇 BGM 先以章節與音訊斷點產生切割草案，使用者試聽確認後才輸出。

## P1：品質與維護

- 下載診斷包預設移除網址權杖、Cookie、使用者路徑等敏感資訊。
- 建立網站適配回歸矩陣、離線假資料測試與重大版本完整歷史雜湊稽核。
- 維持無廣告、免費、低背景資源占用，以及 `current + previous` 的本機版本保留規則。
