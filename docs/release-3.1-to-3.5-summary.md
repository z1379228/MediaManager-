# MediaManager 3.1～3.5 更新簡介

依發行政策，3.1～3.5 不建立獨立 GitHub Release 或重複上傳大型 EXE；以下
功能會併入下一個大版本完整下載，原始碼與測試紀錄仍正常保留。

- **3.1 UI 效能：**轉換與轉錄工作表跳過未變更重繪，並依處理中、閒置、
  隱藏狀態自動調整刷新頻率。
- **3.2 下載可靠性：**加入輸出路徑及 256 MiB 磁碟保留空間預檢，延續原子
  佇列、完成摘要與失敗原因顯示。
- **3.3 YouTube 工作流：**整合搜尋、歷史、相似、替代與試聽停止；本機偏好
  建議必須由使用者點擊，不背景搜尋或自動播放。
- **3.4 長音訊分割：**切割前提示過短片段、低信心切點、預設名稱及時間間隙，
  並顯示安全輸出名稱預覽。
- **3.5 多站品質：**新增不連網的站點能力稽核，涵蓋六個通用網站家族及五項
  Bilibili／彈幕能力，同時要求明列 Cookie、地區、DRM 與付費限制。

完整下載仍使用 `v3.0.0`；上述改善預計隨 `v4.0.0` 或使用者明確指定的大版本
一起提供新的可攜 EXE。
