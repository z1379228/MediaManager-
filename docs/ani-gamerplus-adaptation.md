# AniGamerPlus 參考與動畫瘋 MOD 對照

本文件把 [aniGamerPlus](https://github.com/miyouzi/aniGamerPlus) 的功能拆成可
在 MediaManager 安全邊界內落地的項目。動畫瘋 MOD 只處理官方公開目錄、官方頁面
與使用者已經擁有的本機檔案；不讀 Cookie、不注入腳本、不擷取受保護串流。

## 對照表

| AniGamerPlus 能力 | 動畫瘋 MOD 狀態 | 實作方式／限制 |
| --- | --- | --- |
| 最新、全作品、分類與關鍵字 | 已完成 | 官方目錄快捷入口、屬性／類型／對象／排序篩選、縮圖及搜尋。 |
| 集數導覽與手動選集 | 已完成 | 集數子 MOD 只解析官方作品頁；遇到 Cloudflare 時改由官方頁驗證後貼回 `animeVideo.php?sn=`。 |
| 軟體內開啟官方頁 | 已完成 | Qt WebEngine 僅允許 `https://ani.gamer.com.tw`，無法導向外站。缺少 WebEngine 時保留官方系統瀏覽器入口。 |
| 觀看紀錄 | 已完成 | 本機 bounded JSON，最多 100 筆，只記官方集數 URL 與公開標題；可從選單回到官方頁。 |
| 自動下一集 | 部分完成 | 目前提供使用者選取／開啟下一集；不注入網站腳本，也不在背景自動播放。 |
| 自訂資料夾與檔名 | 部分完成 | 單集離線索引會依作品／集數安全建立資料夾，並可匯入使用者本機媒體；真正串流下載的檔名模板待官方可下載來源確認。 |
| 重試、分段、多執行緒下載 | 待官方下載契約 | 目前解析器只讀公開目錄，不擷取串流；只有取得官方明確允許的下載 URL 且不需 Cookie／DRM／登入時才可接入核心下載佇列。 |
| 1080P、Cookie、Cookie 自動更新 | 排除 | 會涉及帳號憑證、會員／權限或網站限制，不納入安全版本。 |
| 代理、鏈式代理、地區繞過 | 排除 | 不提供繞過地區、存取限制或 Cloudflare 驗證的功能。 |
| 廣告自動跳過、年齡確認自動化 | 排除／手動 | 不自動跳過廣告或代替使用者通過存取提示；官方頁保留手動操作。 |
| FTP、遠端 Dashboard、第三方通知 | 延後 | 會增加外部資料傳輸與憑證管理；目前先使用核心本機下載完成提示與日誌。 |
| 彈幕／留言擷取 | 排除 | 不讀取非必要頁面資料；若日後官方提供公開匯出格式，另立受限子 MOD 評估。 |

## 目前可用的安全組合

1. 啟用「動畫瘋」主 MOD。
2. 依需要啟用「官方搜尋」、「集數導覽」及「番劇儲存」子 MOD。
3. 在官方目錄選取作品，載入集數後可在軟體內開啟官方集數頁，或使用系統瀏覽器。
4. 需要本機保存時，先保存公開索引，再用「匯入／重新連結本機媒體」選取已合法取得的檔案，最後執行 SHA-256 驗證。
5. 「觀看紀錄」只保留本機最近 100 筆，清除應用程式資料即可清除，不會上傳。

## 後續版本優先級

- P0：保持官方網域白名單、Cloudflare 手動驗證、歷史檔案原子寫入與 UI 狀態測試。
- P1：增加離線索引的安全命名模板、使用者選取的下一集操作與本機匯出／清除紀錄。
- P2：只有在取得官方、非登入、非 DRM 的下載來源後，才評估接入可取消、可重試的下載佇列；否則維持目前離線匯入流程。
