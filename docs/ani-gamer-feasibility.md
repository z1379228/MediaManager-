# 巴哈姆特動畫瘋整合可行性檢視

檢視日期：2026-07-16

目前決策：開發版 14.2 來源提供獨立父 MOD、使用者觸發的官方公開目錄／作品搜尋、
封面、分頁集數導覽、官方播放入口，以及只保存公開索引與使用者本機媒體的番劇儲存
子 MOD；這些變更不會回填到既有測試版附件。

## 依據

- [官方服務條款](https://ani.gamer.com.tw/animeRule.php) 說明應透過服務提供的觀看
  功能使用內容，並限制重製及規避廣告、播放或串流限制的行為。
- [官方搜尋頁](https://ani.gamer.com.tw/search.php) 提供公開作品目錄搜尋；
  MediaManager 只解析它回傳的官方作品連結，不存取私有介面。
- [官方動畫頁](https://ani.gamer.com.tw/animeVideo.php?sn=44866) 使用網站自己的播放器與
  彈幕控制；這不代表第三方程式獲准匯出或下載內容。
- [官方播放疑難排解頁](https://ani.gamer.com.tw/anime403info.php) 說明支援環境、IP、代理、
  VPN 與瀏覽器元件等限制；MediaManager 不會規避這些檢查。

## 已提供範圍

動畫瘋功能與 YouTube、Bilibili 完全分開，預設關閉：

- `ani-gamer` 是父 MOD；只有啟用後才建立動畫瘋專用工作區並顯示子 MOD。
- `ani-gamer-search` 負責使用者觸發的官方公開目錄與搜尋。工作區可直接顯示近期熱播、
  新上架、全部動畫及熱門排序，並顯示作品封面；全部動畫篩選只使用固定允許清單內的
  官方屬性、類型、對象與排序參數。
- 目錄與搜尋回應設有大小、筆數、逾時及重新導向限制，只接受動畫瘋 HTTPS 網域與
  數字作品識別碼，不會回退到 YouTube 或其他搜尋來源。
- 當官方搜尋頁要求 Cloudflare 瀏覽器驗證時，provider 會回傳固定的可辨識狀態，
  可信 UI 不顯示底層 403。使用者可在瀏覽器完成官方要求後，將標準
  `animeRef.php?sn=...` 作品網址貼入搜尋框；此路徑不連網即可建立作品選取項目。
- `ani-gamer-episodes` 只在選取作品後讀取官方公開作品頁，擷取固定結構中的數字集數
  識別碼，並以有上限的分頁清單顯示。使用者選取集數後才可開啟官方播放頁。
- 當作品頁同樣要求瀏覽器驗證時，使用者可從官方頁選定單集，再貼入標準
  `animeVideo.php?sn=...` 網址；程式只建立官方入口與離線索引，不匯入 Cookie、
  不模擬驗證，也不解析播放器或串流。
- `ani-gamer-offline` 只有在父 MOD、子 MOD、作品與單集都已明確選取時才可使用。
  它原子保存公開索引及已載入封面，也可由使用者選取自己合法取得的本機媒體，經
  `.part`、取消清理、大小／副檔名限制及 SHA-256 後複製到該單集資料夾。
- 本機媒體匯入不會在啟動時自動執行、不會續跑上次作業，也不會將官方播放頁或只有
  索引的結果標記成已下載影片。
- 群組名稱、工作區文字、狀態與錯誤訊息使用 `en`、`ja`、`zh-CN`、`zh-TW` 四語言
  資源，由核心語言設定統一切換。
- 官方播放入口只接受精確動畫瘋 HTTPS 網域與允許的作品／集數路徑。
- 不在背景連線，不讀取或儲存登入 Cookie，也不啟動下載 provider。

## 明確排除

MediaManager 不檢查私有端點、不擷取或下載動畫瘋串流、不匯入動畫瘋 Cookie、
不匯出彈幕、不移除或自動跳過廣告，也不規避登入、付費、地區、IP、播放器或其他
存取控制。番劇儲存中的媒體只可來自使用者明確選取的本機檔案。

只有在服務公開授權 API、匯出功能或允許第三方離線使用的授權後，才會重新評估下載或
彈幕整合；官方目錄、封面、集數導覽與本機索引不會被解讀成網站下載授權。

## 開發版 16.0 UI 參考模板

16.0 將 FileCentipede 與 iMyFone TopClipper 加入動畫瘋工作區的流程參考，但不安裝、
呼叫或內嵌這兩套軟體：

- FileCentipede 只參考任務詳細資料、佇列、暫停／續傳、進度與完成統計。動畫瘋只會
  把這些模式用於公開索引與使用者自行選取的本機媒體，不擷取網站串流，也不接管
  Cookie、header 或瀏覽器流量。
- TopClipper 只參考「貼網址、辨識成功才啟用下一步、確認選項、加入工作、完成後預覽」
  的分步介面。動畫瘋流程會改為「辨識官方作品／單集、顯示允許動作、建立本機索引或
  預覽使用者本機媒體」，不提供去浮水印、私人內容、地區限制規避或廣告跳過。
- Cloudflare 403 會顯示專用備援區、選取作品上下文及官方單集網址欄位，不把底層
  provider 錯誤或空白集數表當成成功。
