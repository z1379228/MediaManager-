# 開發版 11.0～12.0 更新紀錄

本文件集中記錄 11.x 的實際完成內容；12.0 開始時另建立下一個大版本區間文件。

## 11.0

- 統一網站父 MOD／子 MOD 生命週期：下載、搜尋與宣告式功能使用同一套狀態控制，
  父 MOD 關閉時會停用所有類型的子 MOD，Bootstrap 也會在建立 UI 前修正舊設定。
- Bilibili 下載與彈幕拆分：`bilibili` 負責影片、番劇與分 P，
  `bilibili-danmaku` 獨立控制 XML、ASS 與可選 MKV；搜尋維持
  `bilibili-search` 子 MOD。
- 動畫瘋新增 `ani-gamer` 父 MOD 與專用可信工作區，提供官方近期熱播、新上架、
  全部動畫、熱門排序及屬性／類型／對象篩選；`ani-gamer-search` 維持獨立子 MOD。
- 動畫瘋只開啟允許的官方目錄、作品與播放頁，不下載影片、不擷取串流或彈幕，
  也不處理廣告、登入、地區或播放器限制。
- Facebook 能力限縮為公開影片頁下載：只提供影片格式，不宣告音訊、字幕、分段
  或播放清單能力。
- MEGA 維持獨立公開檔案傳輸工作區，只提供檔案類型辨識、官方 `mega-get` 下載及
  選用速率限制，不與 YouTube／Bilibili 的影音控制混用。
- 十八個內建 MOD、父子群組、四語言資源與新增宣告檔都納入 SHA-256 完整性稽核。
- 新增唯讀 PowerShell 文字污染掃描，可檢查無效 UTF-8、合併標記、工具殘留與
  Unicode 私用區字元，並以非零退出碼阻擋有問題的封裝。
- 本版仍是未簽署開發版，必須以 `SAFE_MODE` 執行；未建立或發布正式版。
