# MediaManager 開發版 6.0 至 7.0 更新紀錄

本文件延續區間式版本紀錄。6.0 是既有整合基線；7.0 專注搜尋工作階段安全、
分頁一致性及來源健康資訊，不改寫 Search v2 的第三方相容語意。

## 6.0 基線

- 有界聯合搜尋、最多 50 筆結果、來源隔離、四語宣告式 MOD UI 與下載容量預檢完成。
- Development／Stable 雙軌版本工具及正式版候選評估器完成。

## 7.0

- provider 的原始分頁 cursor 改由核心包裝成帶 HMAC 的不透明工作階段 token。
- token 綁定查詢文字、內容類型與單一 provider；跨查詢、跨來源及竄改重用會拒絕。
- provider 原始 cursor 不再直接暴露給 UI，仍維持 500 字元及既有分頁能力上限。
- 搜尋來源健康資訊新增連續失敗次數與成功次數；來源恢復後清除錯誤狀態。
- Search v2 合約與既有不支援分頁的 MOD 行為保持相容。

下一階段 MOD 適配更新記錄於 [`release-7.0-8.0.md`](release-7.0-8.0.md)。
