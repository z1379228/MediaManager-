# MediaManager 3.0.0

3.0 將 1.x～2.x 已驗證功能整理為穩定的模組化基準，不新增未經驗證的網站
擷取功能。免費、無廣告、本機優先、預設不遙測與重功能預設關閉等界線不變。

## 本版完成

- 下載佇列改為原列增量更新，下載中、閒置與隱藏頁面採 500／1500／2500 ms
  自適應刷新，避免反覆重建進度元件與畫面閃爍。
- 提供 schema v2 MOD 範本、manifest／套件檢查、API 與 runtime contract 檢查。
- 新增中文第三方 MOD 開發、簽署、安裝、更新、回復與移除流程。
- 新增確定性的核心依賴及授權清單產生器，供凍結與發行稽核使用。

## 驗證結果

- Ruff 通過；完整回歸為 `458 passed, 2 skipped`，略過項目是 Windows 符號連結
  權限案例。
- 1.0～3.0 共 19 個保留版本的版本號、wheel 與 SHA-256 稽核全部通過；3.0
  共有 38 個 checksum 項目。
- `Version/3.0` 的版本、Portable verify、Portable headless 與隱藏 GUI 拷貝
  測試通過，GUI 五秒內保持運行，測試副本已移除。
- 一次性 Ed25519 演練驗證 36 個受保護檔案、可偵測竄改且不保留金鑰。

## 發行狀態

此工作區沒有正式 Ed25519 私鑰與 Windows Authenticode 憑證，因此產出的 3.0
仍是可測試、可攜、完整性可驗證的 `SAFE_MODE` 開發版，不宣稱為正式簽署版。
正式發布時必須在外部受控環境完成雙重簽署與既有 preflight。
