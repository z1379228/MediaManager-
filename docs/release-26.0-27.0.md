# 開發版 26.0～27.0 更新紀錄

## 26.0

- 建立 MOD 契約相容性、範本、權限檢查與子 MOD 分流基線。

## 27.0

- 建立獨立 27.0 Development 來源，不覆寫 26.0 或其他歷史版本。
- 延續動畫瘋嵌入式官方頁面：只允許官方 HTTPS 網域，不注入腳本、不繞過 Cloudflare。
- 強化 MOD 自檢、provider fixture、搜尋結果契約與錯誤診斷。
- YouTube、Bilibili、動畫瘋與 MEGA 維持第一優先；Gopeed、P2P 與遠端控制延後。
- Development 附件只保留本機；對外發布限定為通過驗證的 Testing 版本。
